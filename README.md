### What I learned

- I've learnt how to add a custom font face to my project
- I've also learnt how to set an image's height to be the same as its width

```css
@font-face {
    font-family: 'Outfit';
    font-display: swap;
    font-weight: 400, 700;
    src: url(https://fonts.google.com/specimen/Outfit);
}
img {
  aspect-ratio: 1/1;
}
```
